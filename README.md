# 3D-CAD-modelling-of-Soft-Wearable-Sensor-Sleeve-
Repository focused on the design and prototyping of sensor-based wearable systems for biosignal acquisition from 3D human body reconstructions. It includes anthropometric parameter extraction from STL models, 2D pattern definition, EMG electrode positioning and scaling, and geometry development for sensorized sleeves and upper-limb exoskeletons.
